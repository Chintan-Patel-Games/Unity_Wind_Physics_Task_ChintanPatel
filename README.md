# Realistic Outdoor Environment with Wind & Physics Simulation

A Unity 6 environment simulation project focused on dynamic wind behavior, runtime environmental controls, particle interaction, and wind-based physics systems.

## Features

- Dynamic wind simulation system
- Runtime wind strength and direction controls
- Procedural wind gust system
- Wind-reactive falling leaves particle system
- Wind-reactive physics objects
- Runtime UI controls with sliders and input fields
- Forest environment with terrain, trees, grass, and bushes
- Post-processing and lighting polish
- Fly camera controller using Unity New Input System

## Technologies Used

- Unity 6
- Built-in Render Pipeline
- C#
- Unity Terrain System
- Unity Particle System
- Unity New Input System
- TextMeshPro

## Systems Implemented

### Wind System

- DynamicWindController
- Runtime wind direction updates
- Wind turbulence control
- Procedural gust simulation

### Particle System

- Falling leaves effect
- Wind-driven particle movement
- Rotating leaf particles
- World-space particle simulation

### Physics Interaction

- Wind-reactive rigidbody objects
- Dynamic force application
- Real-time directional interaction

### Runtime UI

- Wind settings panel
- Slider and input field synchronization
- Runtime parameter modification

## Packages Used

- Procedural Terrain Painter | FREE - Automatic Terrain Texturing

The project was optimized to maintain stable runtime performance while supporting:
